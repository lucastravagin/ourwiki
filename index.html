Procedimento de Instalação Software Evigg

Requisitos e Downloads

  • Windows XP, 7, 10 ou +
  • Download da pasta com os arquivos do Software Clique Aqui

Arquivos e Estrutura das pastas

Descrição da estrutura das pastas baixadas no Google Drive para instalação do Software.

	 instalacao-evig/
	 -  Banco de Dados Vazio (Novos Clientes)/ - Banco de Dados vazio para novo cliente
	   	 banco.bak
	 -  BD Strings - Arquivos da String de Conexão dos clientes
		 banco.ini - Arquivo de Conexão do Sql Server
		 bancomy.dll - Arquivo de Conexão do Mysql para controle de licença
	 -  Executável
		 sisnfe.exe
	 -  Licencas
		 cnpj.lic - Arquivo que fica o CNPJ do cliente
		 licenca.lic - Arquivo que o HASH MD5 da licença
	 -  SQL
		  SQLServer2017-SSEI-Expr.exe - Executável do Sql Server
	 -  software.rar
	 -  eVigg.rar

Tabela de Diretórios

Após a realização do download os arquivos deverão ser colocados nas seguintes pastas

Arquivo Colocar no diretório
software.rar Extrair pasta no diretório C:\
eVigg.rar Extrair pasta no diretório C:
banco.ini Colocar arquivo no C:\software\eVigg
software.rar Colocar arquivo no C:\software\eVigg
cnpj.lic Colocar arquivo no C:\software\eVigg
licenca.lic Colocar arquivo no C:\software\eVigg
sisnfe.exe Colocar arquivo no C:\eVigg\SisNFe
